
# FOT
Frame OCR Translate<br>幀緩存光學辨識翻譯器
![示範畫面](https://github.com/Lyciih/FOT/blob/main/images/present1.png)

## instruction
一個基於 XCB 幀緩存截圖的 OCR + 翻譯工具

## 使用方法

### 1 clone
```bash
git clone https://github.com/Lyciih/FOT.git
```

### 2 編譯
```bash
git make
```

### 3 安裝
```bash
git make install
```


### 4 運行一個 server
```bash
git fot -s 
```


### 5 運行 client
```bash
git fot
```

### 6 框選想要翻譯的詞或句子
```bash
git fot
```

### 7 在server觀看結果

### 8 按esc鍵退出client端

### 9 按 ctrl + c 鍵關閉server

### 10 解除安裝
```bash
git make uninstall
```
